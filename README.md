# `Snek Serve`
Snek Serve is a lightweight file server. Features:
* cors support
* directory listing

## Usage
```shell
# Install snekserve globally
npm i -g snekserve
# Run it on the directory you want to serve.
# Then open your browser and visit: localhost:8080
snekserve 8080
```
